# c_exercise

A small collection of C exercises.

## Quick start

```bash
# build (requires clang/gcc + make)
make

# run one exercise (example)
./ex1
```

## Notes

- Files are named `ex*.c` and are intended to be compiled via the provided `Makefile`.
- If you want to build a single file quickly:

```bash
cc -O2 -Wall -Wextra ex1.c -o ex1
```
